Windchill Risk and Reliability Training

Complete Courses

Course # Description Duration
CQ-101

Introduction to Windchill Risk and Reliability (Relex)

Learn about the key features of Prediction, FMEA, RBD and FTA, including data entry, analysis and outputs. Also learn features common to all WR&R modules, including importing, graphing and reporting.

 3 days
CQ-201

Advanced Windchill (Relex) Prediction

Learn about reliability prediction analysis techniques and how to use Windchill Prediction effectively. Gain a solid background in theory with hands-on practice applying the software to real-world examples.

 2 days
CQ-202

Advanced Windchill (Relex) FMEA

Learn the key features of Windchill FMEA, including data entry, analysis and outputs. Gain the ability to completely configure FMEA to your company standards, including the interface, workflow, alerts and advanced FMEA techniques.

 2 days
CQ-203

Advanced Windchill (Relex) FTA

Learn the core concepts of fault tree analysis and how to use Windchill FTA effectively. Gain a solid background in the underlying principles with hands-on practice applying the software to real-world examples.

 2 days
CQ-204

Configuring Windchill (Relex) FRACAS

Learn how to implement a successful closed loop corrective action system by helping you understand many choices, implications, and challenges inherent in establishing an effective FRACAS process, with an emphasis on Windchill (Relex) FRACAS software.

 2 days
